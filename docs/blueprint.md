# **App Name**: Stock Counter

## Core Features:

- Barcode Scanner: Scan the initial and final barcode sequences using the device's camera.
- Sequence Counter: Calculate the number of items by subtracting the initial sequence from the final sequence. Error checking for same or decreasing input.
- Result Display: Display the total count of items on the screen.

## Style Guidelines:

- Primary color: A cool blue (#468189) evoking precision and efficiency. It suggests reliability and trustworthiness.
- Background color: Light grayish-blue (#D6E4E5), desaturated for a clean, unobtrusive backdrop that makes the barcode numbers easily visible.
- Accent color: Muted green (#77ACA2) for confirmation messages or highlighting important actions. This gives the user a sense of satisfaction.
- Body and headline font: 'Inter', a sans-serif font with a modern, neutral look, chosen for readability.
- Simple, clear icons for camera activation and other functions. Consistent design for quick user understanding.
- Clean, intuitive layout with large, easy-to-tap buttons for scanning. Minimize clutter for focus.
